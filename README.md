# 使用

1. 鼠标左键点击并拖动:画线

2. 新建:清空画布(清空背景)

3. 打开:打开图像文件

4. 保存:保存图像文件

5. 鼠标右键点击:清空画布(保留背景)

6. Ctrl+z:撤回

7. Ctrl+y:恢复

8. Ctrl+s:保存

# 打开

目前支持格式:jpg,png,bmp
